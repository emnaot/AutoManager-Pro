# Mini Projet Laravel - Garage de Réparation

## Installation

1. Installer Composer depuis https://getcomposer.org/download/
2. Installer Laravel via Composer
3. Configurer la base de données

## Structure du projet

- API REST pour la gestion des véhicules
- Interface Vue.js pour l'affichage et la gestion
- Base de données MySQL avec XAMPP

## Étapes d'installation

```bash
composer create-project laravel/laravel mini_projet_garage
cd mini_projet_garage
php artisan serve
```
